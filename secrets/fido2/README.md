# FIDO2 Credentials

This directory contains FIDO2-related data for authentication.

## Directory Structure

- `device-credentials/` - FIDO2 credential data per device
- `recovery-codes/` - Backup recovery codes for FIDO2 authentication

## Device Credentials

Each file in `device-credentials/` contains the FIDO2 credential data for PAM authentication.
These files are generated by `fido2-register` and used by pam_u2f.

## Recovery Codes

One-time recovery codes that can be used if the security key is unavailable.
These should be stored securely and regenerated periodically.

## Management Commands

- `fido2-register <username>` - Register a new FIDO2 credential
- `fido2-test <username>` - Test FIDO2 authentication
- Credentials are automatically backed up daily

## Security Notes

- FIDO2 credentials are device-specific and cannot be transferred
- Always have multiple security keys enrolled
- Recovery codes should be stored offline securely
- Monitor the backup logs for any issues
